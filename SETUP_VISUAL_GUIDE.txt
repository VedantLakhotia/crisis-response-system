│ 2. USE IN YOUR COMPONENT:                                                 │
│    ┌──────────────────────────────────────────────────────────────────┐   │
│    │ function MyComponent() {                                         │   │
│    │   const { user, login, logout, isLoggedIn } = useAuth();       │   │
│    │                                                                  │   │
│    │   if (!isLoggedIn) return <LoginScreen />;                     │   │
│    │   return <Dashboard user={user} />;                            │   │
│    │ }                                                               │   │
│    └──────────────────────────────────────────────────────────────────┘   │
│                                                                            │
│ 3. HOOK PROPERTIES:                                                       │
│    • user        → Current logged-in user object (or null)              │
│    • login(data) → Login user: login({ id, role, name, onDuty })       │
│    • logout()    → Logout user & clear storage                         │
│    • isLoggedIn  → Boolean (true if authenticated)                      │
│    • isLoading   → Boolean (checking auth on mount)                     │
